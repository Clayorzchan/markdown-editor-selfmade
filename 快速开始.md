# Markdown Editor - 快速开始

## 🚀 最简单的方式（推荐）

### 1. 先在浏览器中运行

```bash
# 安装核心依赖（跳过 Electron）
npm install react react-dom marked highlight.js react-simple-code-editor prismjs

# 安装开发依赖
npm install -D typescript @types/react @types/react-dom @types/prismjs vite @vitejs/plugin-react

# 启动开发服务器
npm run dev
```

然后在浏览器中打开 `http://localhost:5173`

### 2. 使用启动脚本

```bash
chmod +x start.sh
npm start
```

选择选项 1 可以快速在浏览器中运行。

## 完整功能（Electron 桌面版）

如果需要完整的桌面应用功能（文件保存/打开对话框）：

```bash
# 安装所有依赖
npm install

# 启动 Electron 版本
npm run electron:dev
```
